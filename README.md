# Grid Wall Creator for pyRevit

Wall creation tool from selected grid lines for Revit using pyRevit and Python.

## Overview

Grid Wall Creator generates walls directly from selected grid lines inside Revit.

The tool supports interior and exterior wall workflows and allows wall positioning through adjustable offsets.

## Features

- Create walls from selected grids
- Interior and exterior wall modes
- Side A / Side B control
- Base Offset control
- Adjustable wall height
- Optional automatic wall join
- WPF user interface

Typical workflow:

1. Select grid lines
2. Choose wall mode
3. Set offsets and height
4. Create walls automatically
